A WordPress plugin built with the WPZylos MVC framework.

== Description ==

My Plugin is a WordPress plugin built on the WPZylos framework, providing:

* Modern MVC architecture
* Dependency injection container
* Route-based request handling
* Form validation with sanitization
* Database migrations with dbDelta
* Multi-plugin isolation via PHP-Scoper

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/my-plugin/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Configure the plugin as needed

== Frequently Asked Questions ==

= Does this plugin require PHP 8.1+? =

Yes, PHP 8.1 or higher is required.

= Is this plugin compatible with other WPZylos plugins? =

Yes! Each WPZylos plugin is fully isolated and can run side-by-side without conflicts.
